# Twitch Clone

Un prototype de plateforme de streaming en direct construit avec Next.js et TypeScript.

Résumé

- Application de démonstration pour la gestion de streams, chat en temps réel, et gestion d'utilisateurs.

Prérequis

- Node.js 18+
- Git

Démarrage rapide

1. Cloner le dépôt

```bash
git clone https://github.com/JuniorZ-spec/Twitch-Clone-Project.git
cd Twitch-Clone-Project
```

2. Installer les dépendances

```bash
npm install
```

3. Créer un fichier `.env` à partir de votre modèle et renseigner vos clés secrètes locales.

4. Lancer le serveur en développement

```bash
npm run dev
```
